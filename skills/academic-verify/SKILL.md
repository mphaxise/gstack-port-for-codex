# Academic Verify

Use this skill when the user asks whether a study, citation, or academic claim is trustworthy.

## Workflow

1. State the exact claim being verified.
2. Find the primary publication, DOI, preprint, or official source.
3. Inspect:
   - method
   - sample and data
   - measures
   - limitations
   - conflicts of interest
   - replication or follow-up work
4. Compare against secondary summaries only after reading primary evidence.
5. Return a verdict: supported, partially supported, unsupported, or unresolved.

## Guardrails

- Browse current sources for claims or papers unless the user provides the full text.
- Do not overstate causality.
- Keep quotes short and cite sources.
