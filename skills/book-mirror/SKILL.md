# Book Mirror

Use this skill when the user asks to read a book through the lens of their life, work, or current problems.

## Workflow

1. Confirm the book source is available and allowed to be used.
2. Read chapter by chapter.
3. For each chapter, produce:
   - core idea
   - short compliant evidence
   - connection to the user's projects, values, or decisions
   - practical reflection or action
4. Link relevant brain pages when local context is used.
5. Preserve copyright limits by summarizing rather than reproducing long passages.

## Guardrails

- Do not provide large excerpts from copyrighted books.
- Do not invent user context; search the local brain first.
- Keep the source of each inference clear.
