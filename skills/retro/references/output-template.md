# Output Template

## 1. Summary Metrics

- commits
- contributors
- LOC
- test ratio
- active days or sessions

## 2. Time And Session Patterns

- hourly distribution
- peak hours
- session count and average session length

## 3. Focus And Hotspots

- most-changed files or directories
- focus score or notable context-switching pattern
- ship of the week

## 4. Team Breakdown

- current user first
- then each teammate with praise and one growth opportunity

## 5. Trends

- compare against the previous window when possible

## 6. Next Moves

- what the team should repeat
- what the team should tighten next
