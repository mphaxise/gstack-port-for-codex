# Metrics Checklist

Compute as many of these as the repo can support:

- commits in window
- contributors
- insertions, deletions, and net LOC
- test LOC ratio
- commit time histogram
- work session detection using gaps between commits
- hotspot files and directories
- commit type mix
- biggest ship in the window
- personal and team streaks
- week-over-week or window-over-window deltas when comparing
