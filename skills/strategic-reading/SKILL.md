# Strategic Reading

Use this skill when the user wants a book, article, transcript, memo, or case study read against a specific decision or problem.

## Workflow

1. Define the strategic question.
2. Read the source.
3. Search local brain context for the user's related projects or constraints.
4. Extract:
   - relevant ideas
   - strategic implications
   - risks
   - short-term actions
   - medium-term bets
   - long-term lessons
5. Cite the source and distinguish source claims from your synthesis.

## Guardrails

- Do not summarize the whole source when the user asked for applied strategy.
- Respect copyright limits.
- Do not invent personal context; use brain evidence or state the assumption.
